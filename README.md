# Core - Tienda Online API

Proyecto Spring Boot con arquitectura por capas para gestion de clientes, productos y autenticacion JWT.

## Documentacion

- Guia de excepciones: `GUIA_EXCEPCIONES.md`
- Decisiones de arquitectura (ADR): `docs/architecture-decisions.md`
- Catalogo de clases: `docs/class-catalog.md`
- Estandares de documentacion: `docs/documentation-standards.md`

## Estilo documental profesional

- Lenguaje tecnico claro y consistente.
- Documentacion sin iconografia decorativa ni comentarios informales.
- Responsabilidades por clase claramente definidas.
- En codigo, priorizar Javadoc en capas publicas (`controller`, `service`, `mapper`, `exception`).

Generar Javadoc HTML del proyecto en cada release para publicarlo como artefacto de documentacion interna.
